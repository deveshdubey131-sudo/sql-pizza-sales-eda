# sql-pizza-sales-eda

Pizza Order Delivery Analysis (SQL Project)

Overview

This project focuses on analyzing a pizza delivery dataset using SQL to extract meaningful business insights. The analysis is structured into three levels: Basic, Intermediate, and Advanced, showcasing a progression in data analysis skills.

Dataset

The project uses the following tables:

Orders
Order Details
Pizzas
Pizza Types

Basic Analysis
Total number of orders placed
Total revenue generated
Highest-priced pizza
Most common pizza size ordered
Top 5 most ordered pizza types (by quantity)

Intermediate Analysis

Total quantity of pizzas ordered by category
Distribution of orders by hour of the day
Category-wise pizza distribution
Average number of pizzas ordered per day
Top 3 pizza types based on revenue

Advanced Analysis

Percentage contribution of each pizza type to total revenue
Cumulative revenue generated over time
Top 3 pizza types by revenue within each category

Key Concepts Covered
Joins (INNER JOIN)
Aggregations (SUM, COUNT, AVG)
Grouping (GROUP BY)
Sorting and filtering (ORDER BY, LIMIT)
Common Table Expressions (CTEs)
Window Functions (OVER, PARTITION BY, ORDER BY)
Data formatting and rounding


Conclusion

This project demonstrates how SQL can be used to analyze transactional data and derive actionable insights, moving from basic queries to advanced analytical techniques.
